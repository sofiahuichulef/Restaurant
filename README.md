🍜 UwU Ramen Restaurant - Landing Page

📌 Descripción

Landing Page de un restaurante ficticio llamada UwU Ramen Restaurant, desarrollada como proyecto académico utilizando HTML, CSS, Bootstrap y JavaScript.
El sitio incluye menú interactivo, carrito básico de compras y formulario de reservas con validaciones simples usando manipulación del DOM.

🚀 Tecnologías Utilizadas

HTML5
CSS3
Bootstrap 5
JavaScript (Manipulación básica del DOM)

🎨 Características del Proyecto
✅ Diseño Responsive

Adaptable a dispositivos móviles y escritorio.
Uso de sistema de grid de Bootstrap.

✅ Navbar Funcional

Menú hamburguesa en pantallas pequeñas.
Navegación interna con anclas.

✅ Sección Hero

Imagen representativa del restaurante.
Título y subtítulo.
Botón para navegar al menú.

✅ Menú en Cards

Productos organizados en columnas.
Contador de cantidad por producto.
Botón "Agregar al carrito".

✅ Carrito Dinámico

Actualiza cantidad total en el navbar.
Uso de eventos onclick.
Manipulación del DOM.

✅ Formulario de Reservas

Captura de datos.
Validación básica de campos.
Validación de correo electrónico con expresión regular.
Uso de alert() para confirmaciones.

✅ Footer

Fondo con contraste.
Íconos de redes sociales.
Información básica del restaurante.

🧠 Funcionalidades JavaScript Implementadas

Eventos desde HTML (onclick)
Uso de alert()
Captura de datos de formularios
Validación de email
Modificación dinámica del contenido del DOM
Actualización de contador de carrito
